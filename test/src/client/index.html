Welcome to the Dolphin Maker CLI test environment

How to Test the CLI

  1. First, build the CLI: npm run build
  2. Navigate to this test directory: cd test
  3. Create a static page: node ../dist/cli.js create-page about --type static
  4. Create a dashboard page: node ../dist/cli.js create-page admin --type dashboard
  5. Install dependencies: npm install
  6. Start the dev server: npm run dev
  7. Visit http://localhost:3000/about or http://localhost:3000/admin
